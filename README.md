## Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/marygeraska/to-do-list.git
   ```

2. **Go to the project folder:**
    ```bash
    cd to-do-list
    ```

3. **Install dependencies:**
    ```bash
    npm install
    ```

4. **Run the project:**
    ```bash
    npm run dev
    ```
